Integer range
=============

**ID:** ``file:/jsonschema_extras/schemas/common/range_integer.json``

A range defined by minimum and maximum integer boundaries

Examples
--------

.. code-block:: json

   [
     0,
     100
   ]

.. code-block:: json

   [
     -50,
     50
   ]

.. code-block:: json

   [
     1,
     1000
   ]
